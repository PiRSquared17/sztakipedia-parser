Package hu.sztaki.sztakipediaparser.wiki.converter Description

Provides classes of the converter.

The converter builds and stores the abstract syntax tree during the parsing process. After that it generates the desired HTML output from the tree. It also performs pre and postprocessing and manages CSS classes for the HTML tags.
