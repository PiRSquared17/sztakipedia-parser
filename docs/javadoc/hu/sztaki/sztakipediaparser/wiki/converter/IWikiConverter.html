addInternalLinkTag

void addInternalLinkTag(String url,
                        ArrayList<String> params,
                        String wikitext)
Appends an internal wiki link to the output buffer. e.g. [[Texas|Lone Star State]] Everey link is treated as internal if it begins with double brackets, so File:, Media: and special links are also internal. The method has to figure out which link is which from the url parameter and decide what to do with it.

Parameters:
url - It is not an URL but the name of the article to link to.
params - Parameters of the internal link tag. Can be just an alias or an array of image parameters.
wikitext - The raw wikitext.
